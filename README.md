# iOS-Memo-App

▶︎ CRUD 연습을 위한 iOS 메모앱 만들기

### 기본

| 기능              | 진행상태 | 비고                                |
| ----------------- | -------- | ----------------------------------- |
| 메모 작성(Create) | ✅       |                                     |
| 메모 보기(Read)   | ✅       |                                     |
| 메모 수정(Update) | ✅       | 메모 보기뷰(DetailVC)에서 수정 가능 |
| 메모 삭제(Delete) | ✅       | 오른쪽 스와이프시 메모 삭제 가능    |

### 추가
